# symfony-streaming
This project is builded with symfony 5 for streaming Video/Audio media.

# Streaming video|Audio

## To run this project, you to need to have PHP 8.0 and Nodejs

### Install

```bash
  composer require
  
  yarn install  
```

### Run developement server
```bash
  yarn watch
  
  symfony server:start
```
